# Brainstorm de Design - Landing Page Pesca Pro

## Abordagem 1: Rústico Artesanal & Autêntico
**Probabilidade: 0.08**

### Design Movement
Americana rural com influências de design artesanal do século XX, inspirado em marcas de equipamentos outdoor clássicas (Orvis, Patagonia). Estética "made by hands" com texturas reais.

### Core Principles
1. **Autenticidade Material**: Texturas reais (madeira, lona, metal enferrujado) integradas ao design digital
2. **Tipografia Humanista**: Fontes com caráter, não genéricas—serif para headlines, sans-serif robusto para corpo
3. **Assimetria Intencional**: Layouts não centrados, imagens em ângulos, elementos fora da grade
4. **Narrativa Pessoal**: Cada seção conta uma história, não apenas lista produtos

### Color Philosophy
- **Azul Marinho Profundo** (#1B3A5C): Confiança, profundidade aquática, autoridade
- **Verde Musgo Envelhecido** (#4A6B5C): Natureza intocada, tradição, durabilidade
- **Cinza Quente** (#5A5A52): Neutralidade rústica, madeira envelhecida
- **Acentos em Cobre/Bronze** (#B87333): Warmth, equipamentos metálicos
- **Fundo Creme** (#F5F1E8): Papel envelhecido, conforto visual

**Intenção Emocional**: Confiança ancestral, qualidade comprovada pelo tempo, conexão com a natureza.

### Layout Paradigm
- **Hero**: Imagem assimétrica grande com texto sobreposto em ângulo
- **Seções de Produtos**: Alternância entre imagem à esquerda/direita, nunca centrado
- **Depoimentos**: Timeline vertical com citações destacadas, fotos circulares
- **Rodapé**: Dois colunas assimétricas com informações de contato

### Signature Elements
1. **Linhas Divisórias Orgânicas**: SVG dividers com formas de ondas/peixes, não retas
2. **Badges Artesanais**: Selos/stamps com efeito de carimbo para categorias de produtos
3. **Texturas Sutis**: Padrão de linho/papel no fundo, não flat colors

### Interaction Philosophy
- Hover effects com transições suaves (300ms)
- Botões que parecem físicos (sombra 3D, efeito de pressão)
- Scroll revela elementos com fade-in suave
- Links sublinhados com animação de preenchimento

### Animation
- **Entrance**: Elementos aparecem com fade + slide suave (300ms, ease-out)
- **Hover**: Botões ganham sombra profunda e deslocam 2px para cima
- **Scroll**: Parallax sutil em imagens de fundo (20% de movimento)
- **Depoimentos**: Carousel com transição suave, sem auto-play agressivo

### Typography System
- **Display Font**: Playfair Display (serif elegante) para headlines principais
- **Heading Font**: Crimson Text (serif clássico) para subheadings
- **Body Font**: Inter (sans-serif legível) para corpo e descrições
- **Hierarchy**: 48px/36px/28px/20px/16px com line-height 1.6 para conforto de leitura

## Abordagem 2: Minimalismo Aquático & Moderno
**Probabilidade: 0.07**

### Design Movement
Design nórdico com influências de minimalismo japonês. Foco em espaço negativo, tipografia limpa e fotografia de alta qualidade. Inspiração: Apple meets outdoor brands.

### Core Principles
1. **Espaço Negativo Generoso**: 60% da página é ar, criando elegância
2. **Tipografia Única e Forte**: Uma fonte sans-serif moderna para toda a hierarquia
3. **Fotografia Limpa**: Imagens com composição simples, fundo desfocado
4. **Movimento Sutil**: Animações que não distraem, apenas refinam

### Color Philosophy
- **Azul Profundo** (#0F2B45): Água calma, profundidade, confiança
- **Branco Puro** (#FFFFFF): Clareza, luxo, espaço
- **Cinza Minimalista** (#E8E8E8): Divisões suaves, não preto
- **Acentos em Azul Ciano** (#00BCD4): Energia aquática, CTAs
- **Preto Suave** (#1A1A1A): Texto apenas, não backgrounds

**Intenção Emocional**: Sofisticação, clareza, confiança moderna, foco no essencial.

### Layout Paradigm
- **Hero**: Imagem full-width com texto minimalista sobreposto (esquerda/direita)
- **Seções**: Alternância entre imagem full-width e texto em coluna única
- **Grid de Produtos**: 3 colunas em desktop, cards com espaço generoso
- **Rodapé**: Linha simples, informações alinhadas à esquerda

### Signature Elements
1. **Linhas Horizontais Finas**: Separadores em cinza claro, 1px
2. **Números Grandes**: Estatísticas/benefícios com tipografia enorme (72px+)
3. **Ícones Minimalistas**: Stroke icons em azul, não filled

### Interaction Philosophy
- Hover effects sutis (mudança de cor, não movimento)
- Cliques com feedback visual (ripple suave)
- Transições de página com fade suave
- Scroll revela números/estatísticas com contador animado

### Animation
- **Entrance**: Fade in simples (200ms, ease-in-out)
- **Hover**: Mudança de cor suave, sem movimento
- **Scroll**: Números contam de 0 até valor final (1s, ease-out)
- **Depoimentos**: Fade entre citações, sem movimento horizontal

### Typography System
- **Font Única**: Inter (todas as variações: 700 bold, 600 semibold, 400 regular)
- **Hierarchy**: 56px/40px/28px/18px/14px com line-height 1.5
- **Letter Spacing**: +0.5px em headlines para elegância

## Abordagem 3: Energia Dinâmica & Aventura
**Probabilidade: 0.09**

### Design Movement
Design de esportes radicais/outdoor adventure, inspirado em marcas como GoPro, Red Bull. Cores vibrantes, tipografia bold, movimento constante. Estética "action-packed".

### Core Principles
1. **Cores Vibrantes e Contrastantes**: Paleta ousada que chama atenção
2. **Tipografia Pesada**: Fontes bold/black para impacto visual
3. **Movimento Constante**: Animações em quase todos os elementos
4. **Composição Dinâmica**: Diagonais, sobreposições, quebra de grid

### Color Philosophy
- **Azul Elétrico** (#0066FF): Energia, ação, céu limpo
- **Verde Limão** (#CCFF00): Contraste radical, destaque, vida
- **Preto Profundo** (#0A0A0A): Fundo, contraste máximo
- **Laranja Quente** (#FF6600): Energia, fogo, urgência
- **Branco Brilhante** (#FFFFFF): Clareza, contraste

**Intenção Emocional**: Adrenalina, aventura, energia, conquista, comunidade.

### Layout Paradigm
- **Hero**: Vídeo em background com texto em movimento (parallax agressivo)
- **Seções**: Diagonal cuts entre seções, overlays coloridos
- **Produtos**: Grid 4 colunas com cards que crescem no hover
- **Depoimentos**: Carousel com transição de slide agressiva

### Signature Elements
1. **Diagonal Dividers**: SVG com ângulos de 15-30 graus entre seções
2. **Badges Animados**: Elementos que pulsam ou giram
3. **Padrões Geométricos**: Triângulos, linhas diagonais em backgrounds

### Interaction Philosophy
- Hover effects com movimento (scale, rotate)
- Cliques com feedback explosivo (pulse, bounce)
- Scroll revela elementos com entrada dramática
- Botões com animação de preenchimento (left-to-right)

### Animation
- **Entrance**: Slide + fade com bounce (400ms, cubic-bezier)
- **Hover**: Scale 1.05 + rotate 1-2 graus, sombra profunda
- **Scroll**: Elementos entram com movimento diagonal
- **Depoimentos**: Transição com slide horizontal + fade
- **Números**: Contadores com efeito de "flip" (tipo placar)

### Typography System
- **Display Font**: Bebas Neue (sans-serif bold) para headlines
- **Accent Font**: Montserrat Black (900 weight) para números/CTAs
- **Body Font**: Roboto (sans-serif legível) para corpo
- **Hierarchy**: 64px/48px/32px/20px/16px com line-height 1.4

## Decisão Final
**Abordagem Selecionada: Rústico Artesanal & Autêntico (Abordagem 1)**

Esta abordagem foi escolhida porque:
1. Conecta emocionalmente com pescadores que valorizam qualidade e tradição
2. Diferencia da concorrência (muitos usam minimalismo genérico)
3. Permite storytelling autêntico sobre a loja
4. Paleta de cores rústica é mais memorável e emotiva
5. Texturas e assimetria criam sofisticação sem parecer corporativo

### Documentação de Estilo para Desenvolvimento
- **Cores Principais**: Azul Marinho (#1B3A5C), Verde Musgo (#4A6B5C), Cobre (#B87333)
- **Tipografia**: Playfair Display + Crimson Text + Inter
- **Animações**: Suaves, 300ms, ease-out
- **Texturas**: Linho sutil, linhas orgânicas
- **Princípio**: Autenticidade > Perfeição, Narrativa > Listagem, Assimetria > Centralização
